# List of tests

## Hydrodynamics (possibly with cooling and viscosity)

### 1D cartesian

- Sod shock tube (has analytical solution)
- Woodward & Colella Double blast
- Gaussian pulse advection (has analytical solution)

### 1D spherical

- Sedov blast (has analytical solution)
- Stellar wind (Weaver) (has analytical solution)

### 2D cartesian

- Kelvin-Helmholtz (slab, single interface, different kinds of perturbations (pressure / velocity, eigenmodes / non-eigenmodes)), at different Mach numbers, at different Reynolds numbers

### 3D cartesian

- Turbulent radiative mixing layers (with cooling, e.g. Lancaster 2026)
- Sedov Blast (has analytical solution)
- Stellar wind (Weaver) (has analytical solution)
- Driven turbulence
- Linear waves (has analytical solution)

## Magnetohydrodynamics

### 2D cartesian

- Orszag-Tang vortex
- MHD blast wave
- Jet
- Field loop advection
- Oblique shock tube
- Rotor

### 3D cartesian

- MHD blast wave
- Jet
- Driven turbulence
- Linear waves (has analytical solution)
- Alfven wave
- Field loop advection
- Rotor


## Self-gravity

### 3D cartesian

- Jeans linear waves (has analytical solution)
- Propagating slabs (has analytical solution)
- Spherical collapse

## Cosmic rays

- Cosmic ray modified shocks with and without shock injection (has analytical solution)
